

### 1、本模块主体代码通过代码生成，不推荐修改生成的代码，建议通过业务服务类定制需求。
     @Autowired
     BizUserService bizUserService;
     
     代码生成哈希校验码：[] 

### 2、依赖外部模块的服务必须使用Dubbo注解完成注入。
     @DubboReference
     BizOrgService bizOrgService;     

### 3、业务服务有需要导出的使用Dubbo注解完成服务导出。
     @DubboService
     public class XXService {
          //... 
     }
